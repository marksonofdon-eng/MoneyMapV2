# MoneyMap V2

Standalone marketing site for MoneyMap — bill savings alerts, pricing, and landing pages.

## Requirements

- Node.js 20+
- npm 10+

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open http://localhost:8080

## Production build

```bash
npm run build
npm run preview
```

## Routes

- `/` — Landing page
- `/alerts` — Bill savings alerts
- `/pricing` — Pricing and mission
